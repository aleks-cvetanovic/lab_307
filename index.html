K11-Inhaltsfelder

  • Mit Inhaltsfeldern können zusätzliche Informationen auf engem Raum angezeigt werden. In diesem Kapitel werden vedsadxsawwrschiedene Arten von Inhaltsfeldern vorgestellt und einige davon werden hier aufgeführt. Um diese Inhaltsfelder erstellen zu können, muss erst mal das Skripten unter die Lupe genommen werden.

Ziehharmonika-Anzeige

Wenn auf den Titel in der Ziehharmonika-Anzeige(Akkordeon) geklickt wird, wird das zugehörige Feld erweitert und zeigt seinen Inhalt.

  • Um eine Ziehharmonika-Anzeige erstellen zu können, muss das "<ul>"-Element genutzt werden. Hierzu wird auch noch das "<li>"-Element genutzt. Die "<li>"-Elemente dienen dazu, Einträge in der Ziehharmonika-Anzeige zu erstellen.

    See the Pen qwOYvP by aleks-cvetanovic (@aleks-cvetanovic) on CodePen.

Registerkarten

Wenn man auf einen der Reiter der Registerkarten klickt, wird das zugehörige Feld angezeigt.

See the Pen XQmqQp by aleks-cvetanovic (@aleks-cvetanovic) on CodePen.

Modale Fenster

Das modale Fenster ist eine Box mit Inhalt, welche erscheinen soll, sobald die Webseite geladen ist. Ein solches Fenster muss erst geschlossen werden, damit der Benutzer mit dem Rest der Seite weiterarbeiten kann.
